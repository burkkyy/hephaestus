# hephaestus

## Build Instructions

```sh
cmake .
make
```

## Dependencies

- `cmake`
- `vulkan-sdk`

Install for Ubuntu

```sh
  sudo apt install vulkan-tools
  sudo apt install libvulkan-dev
  sudo apt install vulkan-validationlayers-dev spirv-tools
  sudo apt install libglfw3-dev
  sudo apt install libglm-dev
  sudo apt install cmake
```
